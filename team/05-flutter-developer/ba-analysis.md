## 1. تحليل السوق والمنافسين في المغرب

### السوق المستهدف
- **السوق:** المغرب — قطاع الخدمات المنزلية والحرفية
- **الحجم المقدر:** +2.5 مليون حرفي غير منظم، و +8 مليون أسرة تبحث عن خدمات صيانة سنوياً
- **معدل النمو:** قطاع الخدمات المنزلية الرقمية في شمال أفريقيا ينمو بمعدل 18-22% سنوياً
- **الرقمنة:** أقل من 5% من الحرفيين مسجلون على منصات رقمية — فرصة كبيرة

### تحليل SWOT

| نقطة | تحليل |
|------|-------|
| **Strengths** | استهداف سوق غير مشبع رقمياً، تركيز على معيار ترتيب متعدد الأبعاد، نموذج B2B2C واضح |
| **Weaknesses** | تطبيق جديد بدون قاعدة مستخدمين، تحديات الثقة في الوساطة الرقمية |
| **Opportunities** | تزايد انتشار الهواتف الذكية، تحول السلوك الشرائي إلى الرقمي، دعم الحكومة للاقتصاد الرقمي |
| **Threats** | منافسين محليين ناشئين، صعوبة إقناع الحرفيين بالاشتراكات الشهرية |

### تحليل المنافسين

| المنافس | النموذج | نقاط القوة | نقاط الضعف |
|---------|---------|------------|------------|
| **جيني (Jinni)** | منصة عربية لخدمات المنزل | وجود إقليمي، تطبيق جاهز | ليس مخصصاً للمغرب، أسعار مرتفعة |
| **AlloPneu** | خدمات منزلية مغربية | حضور محلي قوي | خدمات محدودة (سيارات بشكل أساسي) |
| **Moucharik** | منصة مغربية للحرفيين | تمويل حكومي، شبكة واسعة | واجهة قديمة، لا يوجد تطبيق موبايل |
| **بيوت (Bayut)** | عقار وخدمات صيانة | علامة تجارية معروفة | ليس تركيزاً أساسياً على الخدمات الحرفية |
| **عروض مباشرة (Facebook/Groups)** | مجموعات محلية | مجاني، وصول كبير | غير منظم، لا تقييمات، مخاطر عالية |

### الميزة التنافسية لـ Elmokef
1. آلية ترتيب ذكية (مسافة + تقييم + سعر + سرعة استجابة)
2. نموذج اشتراك تدريجي للحرفيين (مجاني → احترافي → مميز)
3. تركيز حصري على السوق المغربي بفهم عميق لخصائصه
4. دعم التقييم والشكاوي لبناء الثقة

## 4. Business Flow Diagram

### 4.1 التدفق الرئيسي: رحلة العميل

```
[فتح التطبيق] → [تسجيل الدخول] → [تحديد الموقع] → [تصفح/بحث خدمات]
       ↓
[عرض قائمة الحرفيين (مرتبة)]
       ↓
[مشاهدة ملف الحرفي] → [الاتصال بالحرفي]
       ↓
[استلام الخدمة] → [تقييم الحرفي + كتابة مراجعة]
```

### 4.2 التدفق الرئيسي: رحلة الحرفي

```
[فتح التطبيق] → [تسجيل حساب] → [إضافة البيانات والوثائق]
       ↓
[اختيار الاشتراك (مجاني/احترافي/مميز)]
       ↓
[إضافة الخدمات والأسعار والصور]
       ↓
[الظهور في نتائج البحث] ← [استقبال طلبات]
       ↓
[تحديث ملفه وإحصائياته]
```

### 4.3 تدفق الاشتراك والمدفوعات

```
[يختار الحرفي الباقة] → [يدخل بيانات الدفع] → [تأكيد الدفع]
       ↓
[ترقية الحساب] → [ظهور أفضل في الترتيب]
       ↓
[تجديد تلقائي شهري] ← [إشعار قبل انتهاء الاشتراك]
```

### 4.4 تدفق نظام الترتيب (Algorithm)

```
[طلب العميل] → [فلترة حسب الخدمة والموقع]
       ↓
[حساب النقاط: المسافة(40%) + التقييم(30%) + السعر(20%) + السرعة(10%)]
       ↓
[تطبيق وزن الاشتراك: المميز +5 نقاط، الاحترافي +2، المجاني 0]
       ↓
[ترتيب تنازلي] → [عرض النتائج]
```

- [x] تسجيل وتسجيل الدخول (Email/Phone/OAuth)
- [x] تحديد الموقع (GPS + يدوي)
- [x] تصفح فئات الخدمات وبحث
- [x] عرض الحرفيين مع الترتيب الذكي
- [x] مشاهدة ملف الحرفي
- [x] الاتصال بالحرفي (هاتف/واتساب)
- [x] تقييم ومراجعة
- [x] المفضلة

#### الحرفي
- [x] تسجيل مع توثيق (رفع هوية)
- [x] إدارة الملف الشخصي (Bio، صور، شهادات)
- [x] إدارة الخدمات والأسعار
- [x] معرض صور الأعمال
- [x] اختيار باقة الاشتراك والدفع
- [x] إشعارات الطلبات

#### الإدارة
- [x] إدارة المستخدمين والحرفيين
- [x] التحقق من الوثائق
- [x] إدارة الفئات
- [x] إدارة الاشتراكات والمدفوعات
- [x] الشكايات
- [x] الإحصائيات الأساسية

### Out of Scope (المرحلة الثانية)
- ❌ بوابة دفع داخلية (استخدام بوابة خارجية)
- ❌ محادثة فورية داخل التطبيق (Chat)
- ❌ حجز مواعيد داخل التطبيق
- ❌ التوصيل المنزلي للمواد
- ❌ نظام ولاء ونقاط مكافآت
- ❌ تطبيق ويب كامل (Admin Panel فقط Web)
- ❌ دفع إلكتروني عبر التطبيق
- ❌ باقة إعلانات للحرفيين
- ❌ تكامل مع وسائل التواصل الاجتماعي خارج نطاق OAuth

### بيانات Dashboard

```dart
// نموذج الـ Dashboard Data (من API)
@freezed
class DashboardData {
  // Stats
  int totalViews;           // 👁️
  int totalContacts;        // 📞
  double avgRating;         // ⭐
  double avgPrice;          // 💰
  int monthlyRequests;      // 📋
  
  // New Requests (آخر 5)
  List<RequestSummary> newRequests;
  
  // Profile completion
  double profileCompletion;  // 0.0 → 1.0
  
  // Recent Reviews (آخر 3)
  List<ReviewSummary> recentReviews;
}

// نموذج الـ Request Summary
@freezed
class RequestSummary {
  String clientName;
  String serviceName;
  double distance;
  String ago;          // "من 5 دقائق"
  RequestStatus status; // new, pending, completed, cancelled
  String? photoUrl;    // صورة العميل (اختياري)
}
```

### أنواع الإشعارات (Notification Types)

| النوع | الأيقونة | اللون | الإجراء |
|-------|---------|-------|---------|
| طلب جديد | 📞 | أخضر | "اتصل" → فتح هاتف |
| تقييم جديد | ⭐ | أصفر | "شوف" → شاشة التقييمات |
| تجديد اشتراك | 💳 | أزرق | "شوف الباقة" → الاشتراكات |
| تذكير بالملف | ℹ️ | رمادي | "أكمل ملفي" → تعديل الملف |

## 4. الـ API Endpoints للـ Dashboard

| الطريقة | الـ Endpoint | البيانات | التحديث |
|---------|-------------|---------|---------|
| GET | `/api/v1/artisans/:id/dashboard` | Stats + latest requests + profile completion | عند فتح الـ Dashboard |
| GET | `/api/v1/artisans/:id/requests?status=new` | قائمة الطلبات الجديدة | عند فتح "طلباتي" |
| GET | `/api/v1/artisans/:id/requests?status=pending` | قائمة قيد الانتظار | |
| GET | `/api/v1/artisans/:id/requests?status=completed` | قائمة الطلبات المكتملة | |
| GET | `/api/v1/artisans/:id/notifications` | آخر 20 إشعار | مع Last-Event-ID للـ SSE |
| PUT | `/api/v1/artisans/:id/requests/:reqId/respond` | قبول أو رفض طلب | عند الضغط على اتصل/رفض |

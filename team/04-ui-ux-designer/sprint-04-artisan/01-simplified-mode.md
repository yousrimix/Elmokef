## 1. ما هو Simplified Mode؟

**المشكلة:** الحرفي غير الرقمي (شخصية حرّاث) يجد الواجهات المعقدة مرعبة. الأيقونات المجردة غير مفهومة، النصوص الكثيرة متعبة، والإحصائيات مربكة.

**الحل:** Simplified Mode هو **وضع عرض بديل** يقدّم واجهة مبسّطة:
- أزرار كبيرة (56px+)
- نصوص قصيرة بالدارجة المغربية
- صور توضيحية بدلاً من أيقونات مجردة
- ألوان عالية التباين
- خطوات قليلة لكل إجراء

### كيف يتم التفعيل؟
- عند تسجيل الحرفي: سؤال "تحب الواجهة المبسّطة ولا العادية؟"
- من الإعدادات: "الواجهة المبسّطة" toggle
- الكشف التلقائي: إذا تأخر المستخدم في التفاعل أو أخطأ مرتين → اقتراح "جرّب الواجهة المبسّطة"

---

## 2. مقارنة: الواجهة العادية vs المبسّطة

| الميزة | الواجهة العادية | Simplified Mode |
|--------|----------------|-----------------|
| حجم الأزرار | 48px | 56px |
| الخط | 16px Body | 20px Body |
| الأيقونات | أيقونات مجردة | صور + أيقونات + نصوص |
| النصوص | عربية فصحى | دارجة مغربية |
| الألوان | عادية | تباين عالي (High Contrast) |
| الإحصائيات | 3 أرقام + رسوم | أيقونات كبيرة + أرقام فقط |
| القوائم | 4 تبويبات في Bottom Nav | 3 أزرار كبيرة في الصفحة الرئيسية |
| التنقل | Bottom Navigation | أزرار كبيرة في Dashboard |

## 4. Simplified Mode — Design Tokens

```dart
// lib/core/theme/simplified_theme.dart
class SimplifiedTheme {
  // أبعاد
  static const double buttonHeight = 56;
  static const double iconSize = 40;
  static const double fontSizeTitle = 24;
  static const double fontSizeBody = 20;
  static const double fontSizeCaption = 16;
  static const double cardRadius = 16;
  static const double avatarSize = 96;
  
  // ألوان (High Contrast)
  static const Color primary = Color(0xFF0D9488);   // أخضر غامق
  static const Color text = Color(0xFF000000);       // أسود (ليس #1A1A1A)
  static const Color bg = Color(0xFFFFFFFF);         // أبيض
  static const Color cardBg = Color(0xFFF0FDF9);     // أخضر فاتح
  static const Color buttonText = Color(0xFFFFFFFF); // أبيض على أخضر
}
```

## 5. مبادئ التصميم لـ Simplified Mode

1. **قاعدة الثلاثة (Rule of 3)**: لا تظهر أكثر من 3 خيارات في الشاشة الواحدة
2. **زر واحد رئيسي**: كل شاشة لها زر CTA واحد فقط
3. **لا قوائم منسدلة**: استبدلها بأزرار أو اختيارات واضحة
4. **صور قبل النصوص**: الصورة التوضيحية أهم من الكلمة
5. **الدارجة المغربية**: النصوص بالدارجة وليس العربية الفصحى
6. **ردود فعل واضحة**: عند الضغط، اهتزاز + تغيير لون + صوت
7. **أخطاء مفهومة**: "هاد الرقم ماشي صحيح" ← وليس "خطأ في الإدخال"

## 4. الـ Star Rating Component (متوافق مع RTL)

### الهيكل

```
[⭐] [⭐] [⭐] [⭐] [☆]
  1    2    3    4    5

في RTL (العربية):
[⭐] [⭐] [⭐] [⭐] [☆]
  5    4    3    2    1
  (يُقرأ من اليمين لليسار — لكن الترقيم ثابت)
```

**ملاحظة RTL:** النجوم نفسها لا تنعكس (تظل من اليسار لليمين بصرياً)، لكن ترتيب القيمة يقرأ من اليمين في العربية. الأهم: الـ 5 نجوم تظل في نفس المكان الفيزيائي.

### مواصفات Flutter

```dart
// lib/core/widgets/star_rating.dart
class StarRating extends StatelessWidget {
  final double rating;         // القيمة الحالية (0.0 – 5.0)
  final int maxRating;         // 5
  final double size;           // 48px (أو 32px للصغير)
  final bool interactive;      // قابل للنقر؟
  final ValueChanged<double>? onRatingChanged; // استدعاء عند التغيير

  // سلوك:
  // - غير تفاعلي: عرض ثابت (في قائمة التقييمات)
  // - تفاعلي: كل نجمة قابلة للنقر
  // - Hover: تغيير لون النجمة المؤقت
  // - Animasyon: scale(1.2) + spring عند النقر
}

// الألوان:
// - نشط: Color(0xFFFBBF24)  (ذهبي)
// - غير نشط: Color(0xFFD1D5DB) (رمادي)
// - Half star: ShaderMask للـ Gradient
```

### حالات الـ Star

| الحالة | المظهر |
|--------|--------|
| Empty ☆ | `#D1D5DB` |
| Filled ⭐ | `#FBBF24` |
| Half ⭐½ | Gradient `#FBBF24` → `#D1D5DB` |
| Hover | `#FDE68A` (أصفر فاتح) + scale 1.1 |
| Pressed | scale 0.95 + spring إلى 1.0 |

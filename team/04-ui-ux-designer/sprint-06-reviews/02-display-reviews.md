## 4. الـ API Endpoints

| الطريقة | الـ Endpoint | البيانات |
|---------|-------------|---------|
| GET | `/api/v1/artisans/:id/reviews?page=&rating_filter=&sort=` | `{data: Review[], pagination: {...}, summary: {avg_rating, total, distribution}}` |
| POST | `/api/v1/reviews` | `{artisan_id, rating, comment}` |
| PUT | `/api/v1/reviews/:id` | `{rating, comment}` — تعديل (خلال 7 أيام) |
| DELETE | `/api/v1/reviews/:id` | حذف التقييم (خلال 7 أيام) |
| GET | `/api/v1/reviews/check?artisan_id=X&client_id=Y` | `{has_reviewed, review_id, can_edit, can_delete}` |
| POST | `/api/v1/reviews/:id/reply` | `{reply_text}` — رد الحرفي |

### تنسيق الـ Review

```json
{
  "id": "rev_123",
  "client_name": "عميد",
  "client_photo": "https://...",
  "rating": 5,
  "comment": "خدمة ممتازة وصل فالوقت المحدد",
  "created_at": "2026-08-31T15:00:00Z",
  "updated_at": null,
  "artisan_reply": {
    "text": "شكراً عميد",
    "created_at": "2026-09-01T10:00:00Z"
  },
  "can_edit": false,
  "can_delete": false
}
```

### توزيع التقييمات (Summary)

```json
{
  "avg_rating": 4.8,
  "total": 32,
  "distribution": {
    "5": 22,
    "4": 7,
    "3": 2,
    "2": 1,
    "1": 0
  }
}
```

## 6. الـ API Endpoints

| الطريقة | الـ Endpoint | البيانات |
|---------|-------------|----------|
| GET | `/api/v1/subscriptions/current` | الاشتراك الحالي + المميزات |
| POST | `/api/v1/subscriptions/auto-renew` | `{enabled: bool}` |
| GET | `/api/v1/payments/history` | سجل المدفوعات مع Pagination |
| GET | `/api/v1/payments/invoice/:id` | تفاصيل الفاتورة |
| POST | `/api/v1/subscriptions/upgrade` | `{new_plan_id}` |

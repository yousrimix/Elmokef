```
┌──────────────────────────────┐

│  ┌──────┐ ┌──────┐ ┌──────┐ │
│  │ 🔧   │ │ 💧   │ │ ⚡   │ │

│  ┌──────┐ ┌──────┐ ┌──────┐ │
│  │ 🎨   │ │ ❄️   │ │ 🔩   │ │
│  │صباغ  │ │تكييف │ │حداد  │ │
│  └──────┘ └──────┘ └──────┘ │
│  ┌──────┐ ┌──────┐ ┌──────┐ │
│  │ 🪚   │ │ 🧹   │ │ 🚗   │ │

│  └──────┘ └──────┘ └──────┘ │

│    [شوف جميع الفئات ←]      │
│                              │
│  ───── حرفيون مقترحون ─────  │
│  [بطاقة] [بطاقة] [بطاقة] ←  │
│                              │
│  🏠  🔍  ❤️  👤              │
└──────────────────────────────┘
```

**العناصر:**
- شريط بحث في الأعلى
- فئات الخدمات (شبكة 3×3 مع أيقونة + اسم)
- قسم "جميع الفئات" ← صفحة الفئات الكاملة
- قسم "مقترح لك" (حرفيون قريبون)
- Bottom Navigation: الرئيسية | بحث | مفضلة | حسابي

**السلوك:**
- نقر على فئة → شاشة 3 (قائمة الحرفيين)
- نقر على بحث → شاشة بحث مع اقتراحات
- طلب الموقع: يُطلب عند اختيار فئة لأول مرة (وليس عند فتح التطبيق)

## شاشة 3: قائمة الحرفيين (مرتبة)

```
┌──────────────────────────────┐
│ ← سباكة        🔍  🗺️  🎛️  │
│                              │
│ "تم العثور على 12 حرفياً"    │
│                              │
│ ┌──────────────────────────┐ │
│ │ [صورة] اسم: يوسف         │ │
│ │        سباك — الدار البيضاء│ │
│ │ ⭐ 4.8 (32 تقييماً)      │ │
│ │ 💰 من 150 درهم           │ │
│ │ 📍 2.3 كم    🕐 يرد بسرعة │ │
│ │                    [اتصل]│ │
│ └──────────────────────────┘ │
│                              │
│ ┌──────────────────────────┐ │
│ │ [صورة] اسم: كريم         │ │
│ │        سباك — الدار البيضاء│ │
│ │ ⭐ 4.5 (18 تقييماً)      │ │
│ │ 💰 من 200 درهم           │ │
│ │ 📍 3.1 كم    🕐 رد خلال 5د│ │
│ │                    [اتصل]│ │
│ └──────────────────────────┘ │
│                              │
│ ┌──────────────────────────┐ │
│ │ [صورة] اسم: حميد         │ │
│ │        سباك — الدار البيضاء│ │
│ │ ⭐ 4.2 (8 تقييماً)       │ │
│ │ 💰 من 120 درهم           │ │
│ │ 📍 5.7 كم    🕐 رد خلال 20د│ │
│ │                    [اتصل]│ │
│ └──────────────────────────┘ │
│                              │
│  🏠  🔍  ❤️  👤              │
└──────────────────────────────┘
```

**العناصر:**
- Header: اسم الفئة + زر بحث + زر خريطة + فلتر
- كل بطاقة حرفي: صورة، اسم، مهنة، تقييم (نجوم + عدد)، السعر (من — إلى)، المسافة، سرعة الرد
- زر CTA: "اتصل"
- Scroll: قائمة Lazy Loading

**الفلتر (عند النقر على 🎛️):**
- نطاق السعر (شريط تمرير)
- المسافة (5km / 10km / 20km / 50km)
- التقييم (3+ / 4+ / 4.5+)
- الترتيب: الأقرب / الأعلى تقييماً / الأقل سعراً

**السلوك:**
- النقر على بطاقة → شاشة 4 (ملف الحرفي)
- النقر على 🗺️ → شاشة خريطة
- النقر على "اتصل" → شاشة 5 (الاتصال)

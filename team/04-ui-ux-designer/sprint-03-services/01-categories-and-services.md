### مواصفات Flutter

```dart
// CategoryGridItem: StatelessWidget
```

### تفاصيل كل بطاقة فئة
| العنصر | القيمة |
|--------|--------|
| الحجم | ~120×120px |
| الخلفية | `#F0FDF9` (أخضر فاتح جداً) |
| زوايا | 12px مدورة |
| الأيقونة | 32×32px, `#0D9488` |
| اسم الفئة | 14px, SemiBold 600, `#1A1A1A` |
| عدد الحرفيين | 12px, Regular, `#6B7280` |

### السلوك
- **نقر على فئة رئيسية** → إذا لها فئات فرعية → **شاشة الفئات الفرعية (CA-02)**
- **نقر على فئة بدون فروع** → مباشرة إلى **قائمة الحرفيين (من Sprint 2: HC-03)**
- **نقر على 🔍** → فتح شاشة البحث (SE-01)

## 6. الـ API Endpoints المطلوبة (لـ محمد)

| Endpoint | البيانات | الاستخدام |
|----------|---------|-----------|
| `GET /api/v1/categories` | `[{id, name_ar, name_fr, icon, image, parent_id, artisan_count}]` | شاشة الفئات الرئيسية |
| `GET /api/v1/categories/:id/subcategories` | `[{id, name_ar, name_fr, icon, artisan_count}]` | الفئات الفرعية |
| `GET /api/v1/artisans?category_id=&subcategory_id=&lat=&lng=&min_price=&max_price=&max_distance=&min_rating=&sort=&cursor=` | `[{id, name, photo, profession, rating, review_count, price_min, price_max, distance, is_verified}]` | قائمة الحرفيين مع الفلترة والترتيب |

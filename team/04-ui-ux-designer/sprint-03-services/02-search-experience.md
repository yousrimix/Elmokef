## 2. مواصفات Flutter للبحث

```dart
// lib/features/search/screens/search_screen.dart

class SearchScreen extends StatefulWidget { ... }

// الـ State:
// - searchController: TextEditingController
// - searchHistory: List<String> (من SharedPreferences — آخر 5)
// - suggestions: AsyncValue<List<String>> (من API — اقتراحات فورية)
// - quickResults: AsyncValue<List<ArtisanSummary>> (نتائج مباشرة)
// - isSearching: bool

// السلوك:
// - عند فتح الشاشة: AutoFocus على SearchBar
// - searchController.addListener مع Debounce (300ms)
//   → بعد 300ms من توقف المستخدم عن الكتابة:
//     → API: GET /api/v1/services/search?q=text → suggestions
//     → API: GET /api/v1/artisans/search?q=text → quickResults
//
// - عند النقر على ✖️: مسح النص + العودة للحالة الافتراضية
// - عند النقر على "إلغاء": Navigator.pop()
// - عند النقر على اقتراح: تنفيذ البحث → قائمة النتائج الكاملة
// - عند النقر على حرفي في quick results: → ملف الحرفي
//
// - حفظ تاريخ البحث:
//   - عند تنفيذ بحث: أضف النص إلى SharedPreferences
//   - حفظ آخر 5 عمليات (إزالة المكرر)
//   - "مسح تاريخ البحث": حذف الكل
```

### الـ API Endpoints للبحث

```dart
// 1. اقتراحات البحث الفورية (تظهر أثناء الكتابة)
// GET /api/v1/services/suggest?q=سباك
// Response: { "suggestions": ["سباكة", "سباك في الدار البيضاء", "سباك رخيص"] }

// 2. البحث عن حرفيين (نتائج سريعة)
// GET /api/v1/artisans/search?q=سباك&limit=3
// Response: { "data": [{id, name, photo, profession, rating, distance}], "total": 12 }

// 3. البحث الكامل (صفحة النتائج)
// GET /api/v1/artisans/search?q=سباك&page=1&filters=...
```

### تردد الـ API Calls

```dart
// Debounce: 300ms
Timer? _debounce;
void _onSearchChanged(String query) {
  _debounce?.cancel();
  _debounce = Timer(const Duration(milliseconds: 300), () {
    _fetchSuggestions(query);
    _fetchQuickResults(query);
  });
}
```

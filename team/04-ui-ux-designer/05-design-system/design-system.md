# 5.5 — Design System: Elmokef
**إعداد:** ليلى السعد — UI/UX Designer  
**الهدف:** توحيد الهوية البصرية — ألوان، خطوط، أيقونات، مسافات، أزرار

---

## 1. الألوان (Colors)

### 1.1 الألوان الأساسية

| اللون | الرمز | HEX | الاستخدام |
|-------|-------|-----|-----------|
| 🟢 الأخضر الرئيسي | --color-primary | `#0D9488` | الأزرار الرئيسية، الشريط العلوي، الروابط النشطة، عناصر الـ CTA |
| 🟢 الأخضر الداكن | --color-primary-dark | `#0F766E` | حالات الـ Hover، الضغط |
| 🟢 الأخضر الفاتح | --color-primary-light | `#14B8A6` | خلفيات التحديد، الـ Highlights |
| 🟠 البرتقالي | --color-accent | `#F59E0B` | دعوة لاتخاذ إجراء ثانوي، التحذيرات، الـ Badges "مميز" |
| 🔴 الأحمر | --color-danger | `#EF4444` | أخطاء، إلغاء، حظر، تنبيهات مهمة |
| 🔵 الأزرق | --color-info | `#3B82F6` | معلومات، روابط، إشعارات |

### 1.2 الألوان المحايدة

| اللون | الرمز | HEX | الاستخدام |
|-------|-------|-----|-----------|
| أسود | --color-text-primary | `#1A1A1A` | النصوص الأساسية |
| رمادي غامق | --color-text-secondary | `#6B7280` | النصوص الثانوية، التواريخ |
| رمادي فاتح | --color-text-muted | `#9CA3AF` | النصوص المعطلة (disabled) |
| خلفية الصفحة | --color-bg | `#F9FAFB` | خلفية التطبيق |
| بطاقة (Card) | --color-card | `#FFFFFF` | خلفية البطاقات |
| حد | --color-border | `#E5E7EB` | الحدود والفواصل |

### 1.3 ألوان التقييم

| اللون | HEX | الاستخدام |
|-------|-----|-----------|
| ⭐ ذهبي | `#FBBF24` | النجوم النشطة |
| ⭐ رمادي | `#D1D5DB` | النجوم غير النشطة |

### 1.4 التباين وإمكانية الوصول

- جميع الألوان تجتاز اختبار WCAG AA (نسبة تباين ≥ 4.5:1 للنصوص العادية)
- الأخضر الرئيسي على الأبيض: نسبة تباين 5.2:1 ✅
- النص الأساسي على الخلفية: 15.4:1 ✅
- النص الثانوي على الخلفية: 6.1:1 ✅

## 2. الخطوط (Typography)

### 2.1 اختيار الخطوط

| الاستخدام | الخط (Arabic) | الخط (French) | الوزن |
|-----------|--------------|---------------|-------|
| العناوين الكبيرة (H1) | Noto Naskh Arabic | Poppins | Bold 700 |
| العناوين (H2-H3) | Noto Naskh Arabic | Poppins | SemiBold 600 |
| النص الرئيسي | Noto Naskh Arabic | Poppins | Regular 400 |
| النص الثانوي | Noto Naskh Arabic | Poppins | Regular 400 |
| الأزرار | Noto Naskh Arabic | Poppins | Medium 500 |
| الأرقام والأسعار | — | Poppins | SemiBold 600 |

**سبب الاختيار:**
- **Noto Naskh Arabic:** خط عربي واضح، يدعم التشكيل، يعمل على جميع الشاشات
- **Poppins:** خط sans-serif عصري ينسجم مع الخط العربي

### 2.2 أحجام الخطوط

| المستوى | حجم الخط (px) | ارتفاع السطر | الوزن |
|---------|--------------|-------------|-------|
| H1 (ترحيب) | 32 | 40px | Bold 700 |
| H2 (عنوان شاشة) | 24 | 32px | SemiBold 600 |
| H3 (عنوان قسم) | 20 | 28px | SemiBold 600 |
| Body Large | 18 | 26px | Regular 400 |
| Body | 16 | 24px | Regular 400 |
| Body Small | 14 | 20px | Regular 400 |
| Caption | 12 | 16px | Regular 400 |
| زر كبير (Button) | 18 | 24px | Medium 500 |
| زر عادي (Button) | 16 | 20px | Medium 500 |
| أرقام الأسعار | 20 | 24px | SemiBold 600 |

## 3. المسافات (Spacing)

| المستوى | القيمة (px) | الاستخدام |
|---------|------------|-----------|
| --space-xs | 4 | تباعد داخلي ضيق |
| --space-sm | 8 | بين العناصر القريبة |
| --space-md | 12 | بين العناصر المترابطة |
| --space-lg | 16 | بين الأقسام، Padding داخلي للبطاقات |
| --space-xl | 24 | بين الأقسام الرئيسية، هامش حواف الشاشة |
| --space-2xl | 32 | فصل بين أقسام كبيرة |
| --space-3xl | 48 | هوامش علوية/سفلية للشاشة |

**شبكة التصميم (Grid):**
- عدد الأعمدة: 4 أعمدة (شاشات ضيقة)
- الهامش الجانبي: 24px
- المسافة بين الأعمدة: 16px

## 4. الأزرار (Buttons)

### 4.1 الأزرار الرئيسية
```
┌──────────────────────────┐
│      Primary Button      │ ← ارتفاع 56px، خلفية خضراء، نص أبيض
└──────────────────────────┘
```

| الحالة | التصميم |
|--------|---------|
| Normal | خلفية `--color-primary`، نص أبيض، زوايا مدورة 12px |
| Hover | خلفية `--color-primary-dark` |
| Pressed | خلفية أغمق، مقياس 0.97 |
| Disabled | خلفية رمادية `#D1D5DB`، نص `#9CA3AF` |

### 4.2 الأزرار الثانوية
```
┌──────────────────────────┐
│     Secondary Button     │ ← إطار 2px، نص أخضر، خلفية شفافة
└──────────────────────────┘
```

| الحالة | التصميم |
|--------|---------|
| Normal | إطار `--color-primary`، نص `--color-primary`، خلفية شفافة |
| Hover | خلفية `--color-primary-light` بنسبة 10% |
| Disabled | إطار رمادي، نص رمادي |

### 4.3 الزر الدائري (FAB)
```
   ┌──────┐
   │  📞  │ ← زر دائري قطر 56px، خلفية خضراء، أيقونة بيضاء
   └──────┘
```

### 4.4 أزرار صغيرة (Chips/Tags)
```
┌──────────────┐ ┌──────────────┐
│  سباكة       │ │  كهرباء      │ ← ارتفاع 32px، خلفية رمادية فاتحة
└──────────────┘ └──────────────┘
```

## 10. التحميل (Loading States)

- **شاشة كاملة:** Animated logo (Splash) + Shimmer effect
- **قائمة:** Shimmer placeholder (3 بطاقات وهمية)
- **زر:** Circular progress داخل الزر، تعطيل النقر
- **صورة:** Placeholder رمادي مع أيقونة صورة + fade-in عند التحميل

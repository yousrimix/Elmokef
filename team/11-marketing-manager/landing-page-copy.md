### 7. الأسئلة الشائعة (FAQ)

```
شنو هو الميقف؟
— تطبيق مغربي كيجمع الحرفيين بالعملاء اللي كيدوّر عليهم.

واش التسجيل فيه فلوس؟
— لأ، التسجيل مجاني. وأول 500 حرفي كيسجلو كياخدو الباقة الاحترافية مجاناً باقي حياتهم.

شنو المحتاج باش نسجّل؟
— سميتك، رقم الهاتف، المدينة، صورة الهوية، و3 صور من أعمالك على الأقل.

شنو الفرق بين الباقة المجانية والاحترافية؟
— الباقة المجانية كتظهر فالقائمة بشكل عادي. الاحترافية كتظهر فالأولوية + طلبات غير محدودة + إحصائيات.

كيفاش كيجيني الزبون؟
— الزبون كيشوف ملفك فالترتيب، كيضغط على رقمك، وكيتصل بيك مباشرة.

شنو الضمان ديالي؟
— التطبيق جديد، ولاكن عندنا نظام تقييمات وشكايات كيضمن الجودة. وأنت فيك الأمر تختار الزبون اللي كيعجبك.
```

## استمارة التسجيل (Form Fields)

```
- الاسم الكامل * (Text)
- رقم الهاتف * (Phone — رقم مغربي)
- المدينة * (Dropdown: الدار البيضاء، الرباط، فاس، مراكش، طنجة)
- الحرفة * (Dropdown: سباك، كهربائي، نجار، صباغ، تكييف، ميكانيكي، حداد، بناء، تنظيف، خدمات عامة)
- سنوات الخبرة * (Number)
- هل لديك صور لأعمالك؟ (Yes/No — إذا لا، يظهر رسالة: "يمكنك رفعها لاحقاً")
- وافق على الشروط * (Checkbox)
- [زر: سجّل مجاناً]
```

## تحسينات لزيادة التحويل (CRO)

| العنصر | التوصية |
|--------|---------|
| **Countdown Timer** | مؤقت تنازلي للعرض المحدود (أيام:ساعات:دقائق) |
| **Live Counter** | عداد حي: "X حرفي سجّل اليوم" |
| **Trust Badges** | شعارات: "مجاني 100%"، "بياناتك آمنة" |
| **Mobile First** | تصميم متجاوب — 90% من الحرفيين يستخدمون الجوال |
| **WhatsApp Click-to-Chat** | زر واتساب ثابت في أسفل الشاشة للاستفسار |
| **Thank You Page** | بعد التسجيل: رسالة ترحيب + خطوات تالية + رابط تحميل التطبيق |
